Fedora 20 Keyboard Shortcuts

Wed 25 February 2015

The Status Quo

Poking around the internet, I couldn't find anything on the keyboard shortcuts in Fedora 20. So, here's a table.

Action Keys Direct Option
Move Window One Workspace Right Shift+Ctl+Alt+Right
Move Window One Workspace Left Shift+Ctl+Alt+Left
Move Window One Workspace Up Super+Shift+Page Up Alt+Ctl+Up
Move Window One Workspace Down Super+Shift+Page Down Alt+Ctl+Down
Switch Applications Super+Tab Alt+Tab (Alt+Esc or Alt+F6)
Switch System Controls Ctl+Alt+Tab Shift+Ctl+Alt+Tab
Move to WorkSpace Up Super+Page Up Ctl+Alt+Up
Move to WorkSpace Down Super+Down Ctl+Alt+Down
Screen Shot Print Alt(Window), Shift(Area), Ctl(Clipboard)
Show Messages Super+M
Focus Notification Super+N
Show Overview Super+S
Show Applications> Super+A
Aplication Menu Super+F10
Show Command Prompt Alt+F2
Log out Ctl+Alt+Del
Lock Screen Super+L
Window Menu Alt+Space
Maximize Super+Up Alt+F10
Minimize Super+Down Alt+F10
Close Alt+F4
Hide Super+H
Move Alt+F7 Alt+F8(Resize)
View Split Super+Left Super+Right

Some Additions

Since I'm used to a very Ubuntu way of doing things, I added in Ctl+Alt+T for the terminal.
